# UCC Course Guide GPT

This project is a custom GPT-based academic assistant designed for the education domain.

## Description
UCC Course Guide GPT helps university students understand course outlines, assignment requirements, grading components, and project expectations using AI.

## Features
- Answer course-related questions
- Explain assignments and grading
- Provide step-by-step project guidance
- Interactive chatbot interface

## Technologies Used
- HTML, CSS, JavaScript
- AnythingLLM
- Docker

## How to Use
1. Open the website
2. Select a suggested question or type your own
3. Click "Try Chatbot"
4. Ask your question

## Note
The chatbot backend runs locally using AnythingLLM and Docker.
