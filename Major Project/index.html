About the Domain

Chosen Domain: Education

UCC Course Guide GPT is a custom academic assistant created for the education domain. It helps university students understand course outlines, assignment requirements, grading components, and project instructions through a simple and interactive interface.

The system is designed to make academic information easier to access and understand, especially when students need quick support while studying or preparing assignments.

How to Use the GPT

  1. Read the suggested questions or type your own course-related question.
  2. Click Copy if you want to copy the question first.
  3. Click Try Chatbot or Open Chatbot.
  4. Paste your question into the chatbot and submit it.
  5. Review the response and verify important deadlines or instructions using official course documents.

Development Process

This project was developed as a custom GPT solution for real-world use in the education sector. The frontend was built using HTML, CSS, and JavaScript to provide a clean and student-friendly interface.

The chatbot functionality was powered through AnythingLLM, while Docker was used to run the environment locally. Academic documents such as course outlines and assignment instructions were uploaded so the assistant could provide relevant answers based on course content.

The interface was also styled using UCC-themed colours to make the system more aligned with the university setting.

Challenges Faced and Solutions Implemented

  • Challenge: Reconnecting the chatbot after restarting the laptop.
    Solution: Restarted the Docker container and reopened the AnythingLLM environment.
  • Challenge: Making the chatbot text more visible and readable.
    Solution: Applied custom CSS and script-based styling fixes to improve text contrast.
  • Challenge: Creating a simple but functional student interface.
    Solution: Added suggested prompts, copy functionality, and quick chatbot access.

Ethical Use and Limitations

This system is intended for academic support only. Students should verify official deadlines, grading rules, and lecturer instructions using Moodle or official course documents.

The assistant may provide incomplete or incorrect answers if the uploaded documents do not contain enough information. It should not be used for plagiarism, cheating, or submitting unreviewed AI-generated work as original work.
